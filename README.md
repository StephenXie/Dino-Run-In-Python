# Dino Run In Python

## How to run
Go to terminal, and run
`python main.py` or `python3 main.py`
## Description
A running game made with pygame
### Goals
Avoid all obstacles by jumping and try to survive 10 levels
### 
## Requirements
You need to have `python 3.x`,`pygame` installed
